WPS JS宏功能以及使用指南

2025-07-11
阅读时长: 3 分钟
浏览量：...

WPS JS宏详细说明及常用表格函数介绍

一、WPS JS宏概述

WPS JS宏是基于JavaScript语言的自动化脚本功能，允许用户在WPS Office（文字、表格、演示）中创建自定义功能，实现办公自动化。与传统的VBA宏不同，JS宏使用更现代的JavaScript语法，具有更好的跨平台兼容性。

JS宏的特点：

  1. 基于JavaScript：使用标准的ES5/ES6语法
  2. 跨平台：在Windows、Linux和Mac上都能运行
  3. 安全性高：运行在沙箱环境中，不会直接访问系统资源
  4. 易于分享：宏代码随文档保存，便于共享

二、JS宏基础使用

1. 启用JS宏

在WPS Office中：

  • 点击"开发工具"选项卡
  • 选择"JS宏"按钮
  • 在弹出的编辑器中编写代码

2. 基本结构

function MyMacro() {
    // 获取当前活动工作表
    let sheet = Application.ActiveSheet;
    
    // 在这里编写你的代码
    sheet.Range("A1").Value = "Hello WPS JS宏!";
}

三、常用表格函数

1. 工作簿和工作表操作

获取活动工作簿和工作表

let workbook = Application.ActiveWorkbook;
let sheet = Application.ActiveSheet;

创建工作表

let newSheet = workbook.Worksheets.Add();
newSheet.Name = "新工作表";

删除工作表

workbook.Worksheets.Item("Sheet1").Delete();

2. 单元格操作

获取和设置单元格值

// 获取A1单元格的值
let value = sheet.Range("A1").Value;

// 设置A1单元格的值
sheet.Range("A1").Value = "新值";

// 设置多个单元格
sheet.Range("A1:B10").Value = "批量填充";

单元格格式设置

let range = sheet.Range("A1:D1");
range.Font.Bold = true;          // 加粗
range.Font.Color = "#FF0000";    // 红色
range.Interior.Color = "#FFFF00"; // 背景黄色
range.HorizontalAlignment = xlCenter; // 居中

3. 常用数据处理函数

查找数据

let searchRange = sheet.Range("A1:A100");
let found = searchRange.Find("查找内容");
if (found) {
    console.log("找到在：" + found.Address());
}

排序

let sortRange = sheet.Range("A1:C100");
sortRange.Sort(sheet.Range("A1"), xlAscending);

筛选

sheet.Range("A1:D100").AutoFilter(1, "条件值");

4. 公式和函数

插入公式

sheet.Range("C1").Formula = "=A1+B1";

常用内置函数

// SUM函数
sheet.Range("D1").Formula = "=SUM(A1:C1)";

// VLOOKUP函数
sheet.Range("E1").Formula = "=VLOOKUP(D1,A1:B100,2,FALSE)";

// IF函数
sheet.Range("F1").Formula = "=IF(A1>100,\"高\",\"低\")";

5. 数据验证

let validationRange = sheet.Range("G1:G10");
validationRange.Validation.Add(
    xlValidateList,             // 验证类型：列表
    xlValidAlertStop,           // 警告样式
    xlBetween,                  // 运算符
    "选项1,选项2,选项3"         // 列表值
);

四、实用示例

1. 批量填充序号

function fillSerialNumbers() {
    let sheet = Application.ActiveSheet;
    let lastRow = sheet.Range("A" + sheet.Rows.Count).End(xlUp).Row;
    
    for (let i = 1; i <= lastRow; i++) {
        sheet.Range("A" + i).Value = i;
    }
}

2. 数据汇总

function summarizeData() {
    let sheet = Application.ActiveSheet;
    let dataRange = sheet.Range("A1:B100");
    let sum = 0;
    
    for (let i = 1; i <= dataRange.Rows.Count; i++) {
        sum += sheet.Range("B" + i).Value || 0;
    }
    
    sheet.Range("C1").Value = "总计";
    sheet.Range("D1").Value = sum;
}

3. 创建表格

function createTable() {
    let sheet = Application.ActiveSheet;
    
    // 添加表头
    sheet.Range("A1:D1").Value = [["姓名", "年龄", "部门", "工资"]];
    
    // 添加数据
    let data = [
        ["张三", 30, "销售部", 5000],
        ["李四", 28, "技术部", 6000],
        ["王五", 35, "管理部", 7000]
    ];
    
    sheet.Range("A2:D4").Value = data;
    
    // 应用表格样式
    let tableRange = sheet.Range("A1:D4");
    let table = sheet.ListObjects.Add(xlSrcRange, tableRange, null, xlYes);
    table.TableStyle = "TableStyleMedium2";
}

五、调试技巧

  1. 使用console.log()：输出调试信息到控制台
  2. 设置断点：在代码行号旁边点击设置断点
  3. 单步执行：使用调试工具栏的按钮逐步执行代码
  4. 查看变量：在调试过程中查看变量值

六、注意事项

  1. JS宏与VBA语法有差异，不能直接混用
  2. 部分Excel VBA功能在WPS JS宏中可能不可用
  3. 宏执行速度可能受数据量影响
  4. 复杂的操作建议分步执行和测试

通过掌握这些WPS JS宏的基础知识和常用表格函数，您可以大大提高办公效率，实现各种自动化处理任务。
