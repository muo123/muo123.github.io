二进制 1 的个数 C++ 实现

2025-06-27
阅读时长: 2 分钟
浏览量：...

题目

题目名称：二进制中1的个数
题目描述：
给定若干个整数（可能包含正数、负数和零），计算每个整数在32位二进制补码表示中1的个数。输入包含多组测试数据，每行一个整数，输出对应整数的二进制中1的个数。

算法思路

核心算法：x = x & (x - 1)
该操作会将整数 x 的最低位1置为0，其他位保持不变。重复此操作直到 x 变为0，操作次数即为1的个数。

算法步骤：

  1. 初始化计数器 cnt = 0。
  2. 当 x ≠ 0 时循环：
    • cnt++
    • 执行 x &= (x - 1)（等价于 x = x & (x - 1)）
  3. 返回 cnt。

算法优势：

  • 高效性：循环次数 = 1的个数（时间复杂度 O(k)，k为1的个数）。
  • 通用性：直接处理负数（利用补码特性），无需额外转换。
  • 简洁性：代码简洁，无复杂边界判断。

关键原理（以8位整数为例）：

操作步骤x（十进制）x（二进制）说明
初始值-211111110最低位1在倒数第2位
x &= (x-1)-411111100清除最低位1
x &= (x-1)-811111000清除下一个最低位1
…继续直到 x=0……共清除31次（32位系统）

样例测试

样例1：正数

  • 输入：5（二进制 0000...0101）
    输出：2
    步骤：
    x=5 (0000...0101) -> cnt=1, x=5&4=4 (0000...0100)
x=4 (0000...0100) -> cnt=2, x=4&3=0
返回2

样例2：负数

  • 输入：-1（二进制 1111...1111）
    输出：32（32位系统）
    步骤：
    需执行32次 x &= (x-1) 才变为0

样例3：零

  • 输入：0（二进制 0000...0000）
    输出：0
    步骤：直接跳过循环。

样例4：边界值

  • 输入：2147483647（INT_MAX，二进制 0111...1111）
    输出：31
  • 输入：-2147483648（INT_MIN，二进制 1000...0000）
    输出：1

代码说明

#include <iostream>
using namespace std;

int bit_count(int x) {
    int cnt = 0;
    while (x) {          // 当x非0时循环
        cnt++;
        x &= (x - 1);   // 清除最低位的1
    }
    return cnt;
}

int main() {
    int n;
    while (cin >> n) {   // 循环读取多组输入
        cout << bit_count(n) << '\n';
    }
}

关键点解释：

  1. 负数处理：
    计算机中负数用补码表示，x - 1 会从最低位1开始“借位”（如 11111110 → 11111101），x & (x - 1) 能正确清除最低位1。
  2. 循环终止：
    当所有1被清除后，x 变为0，循环结束。
  3. 时间复杂度：
    最优 O(1)（如 x=0），最坏 O(32)（如 x=-1），平均高效。

总结

该算法以简洁高效的方式解决了二进制中1的计数问题，巧妙利用位运算特性，统一处理正负数，适合嵌入式系统或高性能场景。
