简单了解 Excel 公式

2025-07-01
阅读时长: 3 分钟
浏览量：...

Excel 公式是用于计算、分析和处理数据的核心工具，由函数、运算符、单元格引用等组成。以下是详细分类和示例：

一、公式基础结构

  • 格式：以等号 = 开头，例如 =A1+B1。
  • 运算符：
    • 算术：+, -, *, /, ^（幂）
    • 比较：=, >, <, <>（不等于）
    • 文本连接：&（如 =A1&" "&B1）

二、常用函数及示例

1. 数学与统计

  • SUM：求和
    =SUM(A1:A10) → 计算A1到A10的和。
  • AVERAGE：平均值
    =AVERAGE(B2:B20) → B2到B20的平均值。
  • COUNT / COUNTA：计数
    =COUNT(A1:A10) → 统计数字单元格数量。

2. 逻辑函数

  • IF：条件判断
    =IF(C1>60, "及格", "不及格") → 若C1>60返回“及格”。
  • AND/OR：多条件判断
    =IF(AND(A1>50, B1<100), "合格", "") → 同时满足A1>50和B1<100时返回“合格”。

3. 文本处理

  • CONCATENATE（或&）：文本合并
    =A1&"-"&B1 → 合并A1和B1，用“-”连接。
  • LEFT/RIGHT/MID：截取文本
    =LEFT(A1, 3) → 取A1左侧3个字符。

4. 日期与时间

  • TODAY/NOW：当前日期/时间
    =TODAY() → 返回当前日期。
  • DATEDIF：日期差
    =DATEDIF(A1, B1, "d") → 计算A1和B1之间的天数。

5. 查找与引用

  • VLOOKUP：垂直查找
    =VLOOKUP("苹果", A1:B10, 2, FALSE) → 在A1:B10区域查找“苹果”并返回对应B列值。
  • INDEX+MATCH：灵活查找
    =INDEX(B1:B10, MATCH("苹果", A1:A10, 0)) → 效果类似VLOOKUP但更灵活。

6. 数组公式（需按 Ctrl+Shift+Enter）

  • SUMIFS：多条件求和
    =SUMIFS(C1:C10, A1:A10, ">50", B1:B10, "<100") → 对A列>50且B列<100的C列求和。
  • SUMPRODUCT：加权求和
    =SUMPRODUCT(A1:A10, B1:B10) → A1:A10与B1:B10对应相乘后求和。

三、高级应用示例

  1. 嵌套IF：多条件判断
    =IF(A1>90, "优", IF(A1>60, "良", "差"))
  2. 动态求和（INDIRECT）：跨表引用
    =SUM(INDIRECT("Sheet2!A1:A10")) → 汇总Sheet2的A1:A10。
  3. 条件格式公式：标记高于平均的值
    =A1>AVERAGE(A$1:A$10) → 应用于A1:A10的条件格式规则。

四、注意事项

  1. 相对/绝对引用：
    • A1（相对引用）：公式拖动时行列会变。
    • $A$1（绝对引用）：固定引用A1。
  2. 错误处理：
    • 使用 IFERROR 屏蔽错误：
      =IFERROR(VLOOKUP(...), "未找到")。

五、实际案例

场景：统计销售表中“产品A”在2023年的总销售额。

=SUMIFS(Sales_Amount, Product_Column, "产品A", Date_Column, ">=2023-1-1", Date_Column, "<=2023-12-31")

通过灵活组合函数和引用，Excel公式能解决从简单计算到复杂数据分析的各种需求。
