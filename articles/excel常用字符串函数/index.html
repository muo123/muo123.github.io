Excel 常用字符串函数详解

Excel 提供了多种强大的字符串处理函数，以下是其中最常用的一些函数及其详细说明和示例：

1. LEN 函数

功能：返回文本字符串中的字符数（长度）

语法：=LEN(text)

示例：

=LEN("Excel")  → 返回 5
=LEN("数据 分析") → 返回 5 (空格也算一个字符)

2. LEFT 函数

功能：从文本字符串的左侧开始提取指定数量的字符

语法：=LEFT(text, [num_chars])

示例：

=LEFT("Microsoft", 5) → 返回 "Micro"
=LEFT(A1, 3) → 返回A1单元格内容的前3个字符

3. RIGHT 函数

功能：从文本字符串的右侧开始提取指定数量的字符

语法：=RIGHT(text, [num_chars])

示例：

=RIGHT("Excel", 3) → 返回 "xel"
=RIGHT(B2, 2) → 返回B2单元格内容的最后2个字符

4. MID 函数

功能：从文本字符串的指定位置开始提取指定数量的字符

语法：=MID(text, start_num, num_chars)

示例：

=MID("abcdefg", 3, 2) → 返回 "cd"
=MID("2023-01-15", 6, 2) → 返回 "01" (提取月份)

5. FIND 和 SEARCH 函数

功能：查找一个字符串在另一个字符串中的位置（区分大小写/不区分）

语法：

  • =FIND(find_text, within_text, [start_num]) (区分大小写)
  • =SEARCH(find_text, within_text, [start_num]) (不区分大小写)

示例：

=FIND("n", "Excel") → 返回 4
=SEARCH("e", "Excel") → 返回 1 (不区分大小写)
=FIND("/", A1) → 查找A1中"/"的位置

6. CONCATENATE 和 & 运算符

功能：将多个文本字符串合并为一个

语法：

  • =CONCATENATE(text1, [text2], ...)
  • =text1 & text2 & ...

示例：

=CONCATENATE("Hello", " ", "World") → 返回 "Hello World"
=A1 & "-" & B1 → 将A1和B1用"-"连接

7. TRIM 函数

功能：删除文本中多余的空格（只保留单词之间的单个空格）

语法：=TRIM(text)

示例：

=TRIM(" Excel 函数 ") → 返回 "Excel 函数"

8. UPPER, LOWER 和 PROPER 函数

功能：转换文本大小写

语法：

  • =UPPER(text) → 全部大写
  • =LOWER(text) → 全部小写
  • =PROPER(text) → 首字母大写

示例：

=UPPER("excel") → 返回 "EXCEL"
=PROPER("john smith") → 返回 "John Smith"

9. SUBSTITUTE 函数

功能：替换文本中的特定字符串

语法：=SUBSTITUTE(text, old_text, new_text, [instance_num])

示例：

=SUBSTITUTE("a-b-c", "-", "/") → 返回 "a/b/c"
=SUBSTITUTE(A1, " ", "", 1) → 只替换A1中第一个空格

10. TEXT 函数

功能：将数值转换为指定格式的文本

语法：=TEXT(value, format_text)

示例：

=TEXT(1234.567, "$#,##0.00") → 返回 "$1,234.57"
=TEXT(TODAY(), "yyyy-mm-dd") → 返回当前日期格式化为"年-月-日"

11. TEXTJOIN 函数 (Excel 2016+)

功能：使用分隔符连接多个文本字符串，可选择忽略空值

语法：=TEXTJOIN(delimiter, ignore_empty, text1, [text2], ...)

示例：

=TEXTJOIN("-", TRUE, "2023", "01", "15") → 返回 "2023-01-15"
=TEXTJOIN(",", FALSE, A1:A5) → 用逗号连接A1到A5的内容

12. REPT 函数

功能：将文本重复指定次数

语法：=REPT(text, number_times)

示例：

=REPT("*", 5) → 返回 "*****"
=REPT("-", LEN(A1)) → 返回与A1内容长度相同的横线

实际应用示例

示例1：从完整姓名中提取姓氏和名字

A1: "张 三"
=LEFT(A1, FIND(" ", A1)-1) → 返回 "张" (姓氏)
=MID(A1, FIND(" ", A1)+1, LEN(A1)) → 返回 "三" (名字)

示例2：格式化电话号码

A1: "13812345678"
=CONCATENATE(LEFT(A1,3), "-", MID(A1,4,4), "-", RIGHT(A1,4)) → 返回 "138-1234-5678"

示例3：清理不规范的文本数据

A1: "  Excel  函数  "
=TRIM(A1) → 返回 "Excel 函数"
=PROPER(TRIM(A1)) → 返回 "Excel 函数"

这些字符串函数可以单独使用，也可以组合使用来解决复杂的数据处理问题。
